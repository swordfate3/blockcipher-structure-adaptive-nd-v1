# Teaching Workspace

All files created by the `$teach` workflow for this repository live under
this directory.

```text
lessons/
├── MISSION.md          teaching goal
├── RESOURCES.md        trusted sources
├── NOTES.md            teaching preferences
├── assets/             shared CSS and interactive scripts
├── reference/          reusable cheat sheets
├── learning-records/   future evidence of demonstrated learning
└── NNNN-*.html         self-contained lessons
```

The only teaching-related records allowed outside this directory are passive
self-improvement records under `.learnings/`.
