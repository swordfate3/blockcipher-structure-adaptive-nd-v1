# Innovation 1 自动化长期目标

## 核心目标

长期推进 `blockcipher-structure-adaptive-nd-v1` 的 Innovation 1 研究，目标是做出真正适配 SPN 类分组密码结构的神经区分器方法。

这个目标不是把某个固定实验跑完，也不是单纯复现或微调已有 baseline，而是围绕 SPN 密码自身结构，持续迭代出更合适的神经网络结构或数据表示方式，让模型能更有效利用：

- S-box cell / nibble 结构
- P-layer / bit permutation 扩散关系
- 差分传播规律
- inverse permutation / state alignment
- candidate trail / transition consistency
- 多 pair 证据聚合
- active pattern 等辅助结构信号

当前主线以 PRESENT/SPN 为主要实验对象，围绕 Zhang/Wang MCND baseline 建立可复现对比，逐步寻找更强、更稳定、更可解释的 SPN 结构适配路线。

## 两类创新方向

### 1. 结构适配神经网络

设计更符合 SPN 拓扑和差分传播规律的模型结构，而不是把 SPN 样本当作普通 bit vector 直接交给通用网络。

可探索方向包括：

- nibble/cell-aware network
- P-layer aligned network
- S-box / DDT-aware network
- trail-aware network
- SPN topology-aware network
- graph / message-passing style SPN backbone
- pair-set evidence aggregation

### 2. 结构适配数据集或特征表示

设计更能暴露 SPN 差分信号的数据组织方式，让网络更容易看到密码结构中的有效证据。

可探索方向包括：

- `DeltaC` 与 `InvP(DeltaC)` 对齐表示
- S-box cell 级输入表示
- P-layer aligned nibble representation
- candidate trail 特征
- transition consistency 特征
- active pattern 辅助特征
- 多 pair 证据池化与统计特征

## 推进原则

每轮推进围绕一个清晰假设展开，避免同时混改模型、数据构造、负样本、metric 和 validation protocol。每个 meaningful 实验都应能回答一个明确问题：

```text
这个变化是否让模型更好地利用了 SPN 结构？
```

实验推进保持基本闭环：

```text
提出假设 -> 写实验计划 -> 本地验证 -> commit/push -> 远程训练 -> 拉回结果 -> 分析归因 -> 更新文档 -> 决定继续、消融、换线或扩大规模
```

执行细节、run id、scale、seed、remote config、gate 和 artifact 路径放在 `docs/experiments/` 的具体计划里，不写进长期目标本身。

研究路线、理论假设、文献理解和方法蓝图放在 `docs/research/`。

## 阶段目标

1. 找到能稳定超过 Zhang/Wang MCND baseline 的 SPN 结构适配信号。
2. 通过多 seed、消融实验和对照实验确认该信号不是偶然波动。
3. 解释收益来自哪类 SPN 结构信息，例如 P-layer 对齐、InvP 表示、S-box cell、DDT prior、trail consistency 或 pair-set aggregation。
4. 把有效路线整理成可复现、可解释、可写入论文的 Innovation 1 方法。

## 阶段切换

出现一次正向结果不代表长期目标结束。正向结果只是进入下一阶段：

```text
探索信号 -> 确认稳定性 -> 做消融归因 -> 形成论文级证据链
```

如果某条路线稳定有效，则继续做 confirmation、attribution、ablation 和 formal evidence。

如果某条路线无效，则停止盲目扩大规模，转向新的 SPN 结构适配假设。

如果当前唯一动作只是等待远程训练结果，则交给 watcher 或子 agent 监控，不在主线程循环轮询。

## 语言边界

在没有足够证据前，不使用“突破”“SOTA”“正式超越”“证明有效”等表述。

更准确的表述包括：

- smoke passed
- diagnostic signal
- paper-scale single-seed positive
- two-seed confirmation evidence
- route-level evidence
- formal route evidence

一句话概括：

```text
目标不是把某个实验跑完，而是迭代出一套真正利用 SPN 密码结构的神经区分器方法。
```
